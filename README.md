Задача 1
Номер 1. Работа со списком (list)

Условие:
Пользователь вводит число элементов, кратное 3, и сами элементы списка. Вывести:

первую треть списка

вторую треть в обратном порядке

последнюю треть в обратном порядке

Алгоритм:

Считываем количество элементов n, проверяем кратность 3.

Считываем n чисел в std::list<int>.

Используем обычный итератор для первой трети, а std::list::reverse_iterator для второй и последней третьей.

Печатаем элементы в требуемом порядке.

Пример (консоль):

Задание 1: Введите количество элементов, кратное 3: 6
Введите элементы списка: 1 2 3 4 5 6
Первая треть: 1 2
Вторая треть в обратном порядке: 4 3
Последняя треть в обратном порядке: 6 5

Задача 2
Номер 2. Вставка элементов в середину дека

Условие:
Пользователь вводит четное количество элементов дека D и сами элементы. Вставить -1 перед каждым элементом второй половины.

Алгоритм:

Считываем четное число n и элементы дека в std::deque<int>.

Получаем итератор на середину дека: it = D.begin() + n/2.

Вставляем -1 перед каждым элементом второй половины через insert.

Печатаем результат.

Пример (консоль):

Задание 2: Введите четное количество элементов дека: 4
Введите элементы дека: 10 20 30 40
Результирующий дек: 10 20 -1 30 -1 40

Задача 3
Номер 3. Вставка первой половины дека в список

Условие:
Есть список L и дек D. Вставить первую половину дека в начало списка, вторую половину удалить из дека.

Алгоритм:

Считываем список и дек.

Используем insert(L.begin(), D.begin(), D.begin() + n/2) для вставки первой половины.

Удаляем первую половину из дека erase(D.begin(), D.begin() + n/2).

Печатаем список после вставки.

Пример (консоль):

Введите количество элементов списка L: 2
Введите элементы списка: 5 6
Введите четное количество элементов дека D: 4
Введите элементы дека: 1 2 3 4
Список L после вставки: 1 2 5 6

Задача 4
Номер 4. Запись ASCII-кодов в файл

Условие:
Пользователь вводит строку и имя файла. В файл записываются двойные ASCII-коды символов (символ * 2).

Алгоритм:

Считываем строку и имя файла.

Через std::ofstream записываем int(c) * 2 для каждого символа.

Закрываем файл и выводим сообщение.

Пример (консоль):

Задание 4: Введите имя файла: out.txt
Введите строку символов: AB
Данные записаны в файл out.txt


Файл out.txt содержит:

130 132

Задача 5
Номер 5. Дублирование элемента списка по вектору

Условие:
Пользователь вводит четный вектор V и список L. Найти последний элемент списка, который есть в первой половине вектора, и дублировать его после самого себя.

Алгоритм:

Считываем вектор и список.

Находим последний элемент списка, присутствующий в первой половине вектора (std::find).

Дублируем его в списке (insert).

Печатаем результат.

Пример (консоль):

Введите четное количество элементов вектора V: 4
Введите элементы вектора: 1 2 3 4
Введите элементы списка L через пробел (Enter для окончания): 5 2 6
Список L после дублирования: 5 2 2 6

Задача 6
Номер 6. Удаление нулей из второй половины вектора

Условие:
Удалить все нули во второй половине вектора V.

Алгоритм:

Считываем четный вектор.

Применяем std::remove к диапазону второй половины.

Удаляем "хвост" через erase.

Печатаем результат.

Пример (консоль):

Введите четное количество элементов вектора V: 6
Введите элементы вектора: 1 2 0 3 0 4
Вектор после удаления нулей во второй половине: 1 2 0 3 4

Задача 7
Номер 7. Сортировка второй половины и слияние с первой

Условие:
Первая половина вектора отсортирована. Отсортировать вторую половину и объединить с первой.

Алгоритм:

Сортируем вторую половину через std::sort.

Используем std::inplace_merge для слияния двух отсортированных частей.

Печатаем итоговый вектор.

Пример (консоль):

Введите четное количество элементов вектора V: 6
Введите элементы вектора (первая половина уже отсортирована): 1 3 5 4 2 6
Вектор после сортировки и слияния: 1 2 3 4 5 6

Задача 8
Номер 8. Формирование деки из первых и последних букв соседних слов

Условие:
Пользователь вводит список слов. Для каждой пары соседних слов формируется строка из первой буквы первого слова и последней буквы второго слова. Эти строки помещаются в дек std::deque.

Алгоритм:

Считываем список слов std::list<std::string>.

Проходим парами соседних элементов через итераторы.

Формируем строку (*it1)[0] + (*it2).back() и добавляем в дек.

Печатаем дек.

Пример (консоль):

Введите количество слов в списке L: 4
Введите слова: словарь программа окно мышь
Дек D: са по он


Примечание:
Для русских букв корректнее использовать std::wstring и std::wcout, иначе берётся только первый байт UTF-8.
